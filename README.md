# משנה תורה — Mishneh Torah Visual Study Guides

Visual slide-based study guides for the Rambam's Mishneh Torah, following the daily study cycle.

## Structure
- **14 ספרים** (Books)
- **83 הלכות** (Sections)  
- **~1,006 פרקים** (Chapters)

Each perek includes:
- 9 visual slides covering the chapter's key halachot
- Hebrew text with English translation
- Quiz for review
